***** 技术学习建议
1. 学习技术的建议
    - 学 TCP/IP，把 Nmap 和 Hping 的手册看一遍。
    - 学 HTTP，把 Curl 的手册看一遍。
    - 学 SQL 注入，把 SQLmap 的手册看一遍。
    - 学 WEB 前端安全，把 BeEF 的手册看一遍。
2. 建议
    - API 的“指令类参数”，一定要和其它参数分割开来，最好是独立的参数，并且只能接受常量。比如格式化串 API 的格式参数，不接受变量。
